# homebrew-tap
Taps for da Apps

## Install

```bash
brew tap erewhon/tap
```

## Formulas

| Formula | Description | |
|---------|-------------|---|
| [agent-monitor](https://github.com/erewhon/agent-monitor) | TUI for tracking Claude Code agents in tmux sessions | `brew install erewhon/tap/agent-monitor` |
| [gaol](https://github.com/erewhon/gaol) | Cross-platform container and VM management tool | `brew install erewhon/tap/gaol` |
| [protectinator](https://github.com/erewhon/protectinator) | Portable, zero-config security monitoring tool | `brew install erewhon/tap/protectinator` |

## Casks

| Cask | Description | |
|------|-------------|---|
| [nous](https://github.com/erewhon/nous) | Personal notebook and data organization tool | `brew install --cask erewhon/tap/nous` |
